Declaración recomendada para la creación de arrays en JavaScript pop() y push()

var soyUnArray = [];

  • Utilidad

    Pop sirve para eliminar el ultimo elemento del array y Push para añadir un elemento al final del array.

  • ¿Modifican el array original?

    Si, lo modifica

  • ¿Devuelven algún valor?

    Push en caso de que añade algo devuelve 1 y en caso contrario 0. Pop devuelve el elemento que a eliminado

shift() y unshift(),

  • Utilidad

    Shift elimina el primer elemento del array y unshift añade los elementos que se le pasen por parámetro al principio del array.

  • ¿Modifican el array original?

    Si, lo modifican

  • ¿Devuelven algún valor?

    Shift devuelve el elemento que a eliminado y unshift devuelve 1 en caso de haber introducido el elemento en el array y 0 en el caso contrario

De los cuatro métodos anteriores, indica los recomendados para la inserción/eliminación de un elemento y por qué

Se recomienda pop y push ya que son más rápidos al no tener que modificar las posiciones de el resto de elementos.

join() y split(), ¿son métodos de String o de Array?

Join es de array y split es de string

splice()

  • Utilidad

    Sirve para añadir o eliminar elementos de un array, sustituyendolos por los indicados en los parámetros del método.

  • Indica significado del primer argumento admitido: índice, cantidad o contenido

    Índice

  • Indica significado del segundo argumento admitido: índice, cantidad o contenido

    Cantidad

  • Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

    Contenido

  • ¿Modifican el array original?

    Si, lo modifican

slice()

  • Utilidad

    Slice es un método que devuelve una selección de elementos de un array

  • Indica significado del primer argumento admitido: índice, cantidad o contenido

    Índice
  • Indica significado del segundo argumento admitido: índice, cantidad o contenido

    Índice

  • Indica significado del resto de argumentos admitidos: índice, cantidad o contenido

    No tiene tercer argumento

  • ¿Modifican el array original?

    No se modifica el array original, devuelve otro con el resultado.
